Installtion
================

1. install Python 2.7

2. install wxPython for Python 2.7

3. Install required packages

  > easy_install pil
  > easy_install markdown 
  > easy_install cheetah
  > easy_install reportlab
  > easy_install html5lib
  > easy_install pisa

4. Copy _namemapper.pyd to site-package\cheetah-*.egg\Cheetah
   - download proper package from http://www.lfd.uci.edu/~gohlke/pythonlibs/#base
   - open the package with zip program(ex: 7-zip) and extract _namemapper.pyd
