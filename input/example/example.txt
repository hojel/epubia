# 이상한 나라의 앨리스
19세기 Lewis Caroll이 이웃 앨리스 루델을 위해 쓴 판타지 소설.
이후의 많은 판타지들에 영감을 주었다.

하얀토끼를 좋아 지하세계로 들어가면서 겪는 앨리스의 이야기로 
다음과 같은 등장인들들도 유명하다.

* 채셔고양이
* 미친 모자장수
* 3월의 토끼
* 가짜거북
* 빨간여왕
* 도마뱀 빌

주인공 앨리스보다	
채셔고양이, 미친 모자장수, 하얀토끼	
삼인방이 더 유명하다.

두번째 앨리스의 이야기로 이번엔 거울나라로 들어가면서 겪는 이야기이다.
등장인물이 줄어든 대신 체스를 바탕으로 이야기가 진행된다.
